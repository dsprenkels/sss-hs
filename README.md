These are the Haskell bindings for my Shamir secret sharing library.

# Install and build

```shell
# Clone and cd into the source code
git clone --submodule https://github.com/dsprenkels/sss-hs.git
cd sss-hs

# Install the library using Cabal
cabal install
```
